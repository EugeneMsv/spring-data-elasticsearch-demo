# Getting Started

### Reference Documentation

Demo project for spring-data-elasticsearch.

Requirements:
* java 11
* docker

How to run:
* Go to root project folder.
* Run 
  ```
    mvnw spring-boot:build-image
  ```
* Run 
  ```
    docker-compose up
  ```
* Import to postman and try api
  `
  Elasticsearch-demo.postman_collection.json
  `
